## Structural finding (2026-06-05) — why real-data vindication keeps failing

The move-4 real-data hunt swept ~8 substrate families (skyrmion · quantum-Hall/QAH · topological photonics · fully-frustrated Josephson · semiconductor ring laser · magnetic vortex · plus the bistable-memory class). It converged on **one law**, not a list of near-misses:

> **The TDT needs a *transverse* circulation — a current in a different sector than the escape coordinate — sustained by an affinity *decoupled* from the metric barrier.** That is a property of **driven-NESS-with-fuel** (triad, glass, QEC, the DNA network). Real devices structurally lack it, in exactly three ways:

1. **No current** — static-state memory (MRAM/MTJ, SRAM, single-domain bits) stores a *metric coordinate* (magnetization/voltage). Its retention-vs-bias r(δ) data is pristine but is plain biased Kramers escape; there is no current to be decoupled from. Fails (i).
2. **Collinear current** — current-state memory (magnetic vortex ✓ confirmed; flux qubit, ε = 2I_pΦ; *suspected* for the ring laser) stores the bit **as the current's direction**, so the escape *is* the reversal of 𝒜. Same DoF, not transverse. Fails move-1.
3. **Co-tuned current** — single-knob driven oscillators (ring-laser pump, parametric pump) crank 𝒜 and the barrier with one knob. Fails move-2.

**The device-inversion proves its own limit.** A device that "needed to measure r(δ)" needed it *because it is a memory*; a memory is bistable storage; bistable storage makes the current the bit (collinear) or absent. So no spec-sheet r(δ) device can carry the transverse decomposition — by construction. And **r(δ) ≠ the TDT selection rule**: r(δ) is biased escape (Kramers, universal); the TDT is ∂r/∂𝒜 gated by δ, which *requires the current as an independent axis*.

**Consequence.** Real-data vindication, if pursued, is a *commissioned driven-circulation measurement* (the Josephson-array / ring-laser proposals — the ring laser's own transverse-vs-collinear status pending the (θ,ψ) derivation), **not** an archive question. The corrected *archive* search is the driven-NESS-with-transverse-circulation class — driven chemical/biochemical reaction networks and driven colloidal/single-particle NESS (the physical realization of the glass ring) — see `docs/research_prompt_driven_ness_transverse.md`.
