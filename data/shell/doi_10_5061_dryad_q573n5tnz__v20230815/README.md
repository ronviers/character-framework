# README

GENERAL INFORMATION

Article: Evolutionary process of extremely twisted heteromorph ammonites from the Upper Cretaceous in Japan

Authors: Akihiro Misaki, Takashi Okamoto and Haruyoshi Maeda

Journal: Papers in Palaeontology

Specimens: Morphotypes I–VIII of Didymoceras morozumii, D. awajiense and Pravitoceras sigmoidale from the Upper Cretaceous Izumi Group and the Toyajo Formation (Sotoizumi Group) in study areas A (around Mt. Toyajo in the Kii Peninsula) and B (region from northeastern Shikoku to southwestern Awaji Island) in south-west Japan

DATA

Data 1. Material information. Register number (museum register number of the specimen), species (whether Didymoceras morozumii, D. awajiense or Pravitoceras sigmoidale), morphotype (whether I, II, III, IV, V, VI, VII or VIII), horizon (whether A, B, C or D), study area (whether A or B), section (whether 1, 2, 3, 4, 5, 6 or 7), locality (detailed locality), reference (studies that have examined the same specimen), coiling direction (whether dextral or sinistral) and remarks (short explanation of the specimen) of studied specimens are shown.

Data 2. Parameter values of the growing tube model for each morphotype. The details of this model and parameters are explained by Okamoto (1988). E, radius enlarging rate; C, standardised curvature; T, standardised torsion; s, growth stage parameter. The E, C and T values for each s value of all data points are shown for each morphotype.
